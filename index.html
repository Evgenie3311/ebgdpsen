FAQ - What is EBGDPS?

EBGDPS is a private server for the game Geometry Dash. Our server is under development and we will do our best to release updates.

Where and how do I download the server?

You can download the game from the button below, as well as register you can also use another button below. If you download a server for Android, you simply download the game and run it. If you are downloading a server for Windows, you need to unzip the archive, move it wherever you want, create an EBGDPS.exe file shortcut and move it to the desktop

Warning!

If you're on PC: Change graphics in settings to medium for see custom textures, icons and other.
